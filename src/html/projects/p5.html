Description :

Ce projet a été réalisé en collaboration avec Thales dans le cadre de la SAÉ 15. L’objectif était de concevoir un système complet permettant de prendre des photos de bords de test à l’aide d’un microcontrôleur Pico et d’un Raspberry Pi, puis de les afficher automatiquement dans une galerie web sécurisée. Nous avons développé une application web avec : Un système de gestion des comptes utilisateurs hiérarchisé (utilisateur simple, modérateur, super administrateur), avec des droits d’accès et d’action différenciés. Une interface de prise de photo pilotée à distance via le Raspberry, intégrant les images capturées dans une galerie dynamique. Une base de données pour gérer les utilisateurs, les images, et les journaux d’activité. Ce projet m’a permis de renforcer mes compétences en développement web full-stack, en programmation embarquée et en gestion des droits d’accès. J’ai aussi appris à travailler en mode projet avec un partenaire industriel, avec des contraintes de fiabilité et de sécurité.
