- SAÉ11 -

Description :

Dans le cadre de cette SAE, notre groupe devait rechercher et analyser plusieurs types de cyberattaques. Nous avons choisi d’en détailler trois précisément : une attaque de phishing (hameçonnage), une injection SQL et une attaque de type ransomware. Pour chacune, nous avons décrit le déroulé de l’incident, les failles exploitées et les mesures mises en place par les équipes de sécurité pour y remédier.
