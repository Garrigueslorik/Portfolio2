Projet Thales

- SAÉ15 -

Description :

Dans cette seconde phase du projet réalisé avec Thales, l’objectif était de concevoir et développer intégralement le système fonctionnel permettant la prise de photos à distance via un Raspberry Pi et un Pico, puis leur affichage sécurisé sur un site web dynamique. J’ai participé à l’ensemble de la chaîne technique : Développement web complet en PHP, HTML et CSS, avec gestion des rôles utilisateurs (admin, modérateur, utilisateur simple) et système de connexion sécurisé. Création et structuration d’une base de données (MySQL) pour gérer les utilisateurs, les logs, les images et les droits. Une mauvaise structuration empêchait le fonctionnement correct du système d’authentification et de la gestion de session. Scripts Python embarqués pour piloter le Pico (capture de photos) et communiquer avec le Raspberry Pi, en assurant l'envoi des fichiers vers le serveur. Mise en place de la liaison entre la base de données et l’interface web via PHP, pour permettre l’enregistrement et l’affichage dynamique des images dans la galerie. Intégration du stockage des logs, avec visualisation possible par les utilisateurs habilités. Ce projet m’a permis d’approfondir mes compétences en : Développement web back-end et base de données relationnelle, Programmation embarquée et interactions entre systèmes physiques et logiciels, Architecture client-serveur, Travail rigoureux en conditions semi-professionnelles avec un partenaire industriel.
